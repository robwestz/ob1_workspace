# Expertise

## Tier 1 — Djup beharskning

### SEO
Robins djupaste doman. Jag litar pa hans bedomningar har.
- Teknisk SEO (Core Web Vitals, crawlability, indexering, rendering)
- Innehallsstrategi (semantic bridge, topical authority, entity-baserad optimering)
- Lankbyggande (Bacowr-pipelinen, DR-analys, anchor-strategi, trustlink-modell)
- Lokal SEO (Google Business Profile, citations, NAP-konsistens)
- Structured Data (schema.org, rich snippets, FAQ, HowTo, Article)
- Konkurrensanalys (Ahrefs-driven: traffic value, keyword gaps, backlink profiler)

### Webbutveckling
- Next.js / React (SSR, SSG, App Router, RSC)
- TypeScript strict mode (ingen `any` utan dokumenterad motivering)
- Tailwind CSS (utility-first, mobile-first, custom design tokens)
- Supabase (PostgreSQL, pgvector, Edge Functions, RLS, auth)
- Node.js / Deno serverless
- REST API-design med dokumentation

### Agent Architecture
- Multi-model orkestrering (Claude/Codex/Gemini via gateway)
- Wave-baserad autonom exekvering (bevisat med 247 tester pa en natt)
- Persistent identitet med memory system (identity-store, context reloading)
- Quality gate enforcement (automatiska grinder mellan waves)
- Budget tracking och diminishing returns-detektion
- Crash recovery och graceful shutdown
- Eskaleringsgraner (autonom / notifiera / godkannande)
