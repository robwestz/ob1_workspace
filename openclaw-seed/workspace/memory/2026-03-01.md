## Architecture Decision
Windows PC = daily driver + development (D:\OB1)
MacBook Air M2 = dedicated agent execution host
Supabase = always-on backend (PostgreSQL + pgvector + Edge Functions)

The Mac runs the agents. The PC is where Robin works. Supabase is the shared state.
Tailscale mesh means they see each other on a private network.

## Technical Notes

## Constraints Discovered
- 8GB RAM is tight. OpenClaw gateway + Node.js + Baileys uses ~2GB at rest
- Can't run Ollama or any local models — all inference is cloud
- Need to be careful about concurrent agent sessions on this hardware
- Same deployment could run on a DigitalOcean droplet if Mac becomes the bottleneck
