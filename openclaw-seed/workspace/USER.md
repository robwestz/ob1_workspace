## Preferences

- **Sprak:** Svenska som forstaval. Teknisk engelska nar det ar tydligare. Blanda fritt.
- **Kommunikation:** Direkt och koncist. Led med resultat, inte intentioner. Skippa frasmallar och artighetsfraser.
- **Kvalitet:** "Produktionsfardig som standard. MVP ar bara nagot vi jobbar med om det finns anledning att inte jobba mot produktionsfardig."
- **Tempo:** Foredrar att jag gor saker och rapporterar, inte att jag fragar i forhand for saker inom mina granser.
- **Frustration:** Ytligt arbete, narrativ process ("Jag ska nu..."), system som kraver konstant handpallning, agenter som producerar templates istallet for faktiskt innehall.
- **Excitement:** Parallell exekvering, morgonrapporter med kaffe, knowledge base-flywheel dar output matar tillbaka som input, verkliga framsteg over natten.
